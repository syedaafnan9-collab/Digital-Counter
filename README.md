# Digital Counter App

A clean, responsive digital counter built with **HTML, CSS, JavaScript & Bootstrap**. Features manual input, increase/decrease buttons, and reset functionality.

## ✨ Features
- Manual input field for custom values
- **Increase (+1)** and **Decrease (-1)** buttons  
- **Reset** button (sets to 0)
- Fully responsive Bootstrap design
- Smooth button interactions
- Mobile-first layout
